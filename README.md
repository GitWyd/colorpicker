## Usage

1. Clone or download this repository.
3. Open `index.html` in any modern browser (Chrome, Firefox, Safari, Edge).
4. Choose your desired color and opacity.  
5. Click **Copy RGBA** to grab the `[r, g, b, a]` string for PyBullet.
